# Findings

- Multiple executable files were identified on the USB device during forensic examination.
- Several files exhibited mismatched file extensions and signatures, indicating potential attempts at obfuscation or file masquerading.
- Hash analysis revealed that at least one executable matched known malicious indicators referenced in a recent threat intelligence report.
- Timestamp analysis showed that suspicious files were introduced within a narrow timeframe consistent with reported abnormal activity.
- Metadata and security identifier analysis linked the USB device activity to a specific employee account within the Sales and Marketing department.
- No evidence was found indicating alteration of the forensic image during analysis, as all hash values matched initial acquisition values.
